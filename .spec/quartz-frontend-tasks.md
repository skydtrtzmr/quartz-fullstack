# Quartz 前端研发任务清单

- [ ] Quartz前端-图谱-边显示优化：增加记录边名称（追溯边的来源类型，显示正文引用或YAML属性），供图谱显示；增加有向边箭头。
- [ ] Quartz前端-图谱-构建时优化：构建时预计算每个页面的局部图谱关联节点，按节点ID拆分为多个JSON文件，减少运行时运算量，优化首屏加载速度。考虑文件数量过多时，需采取分片（按首字母或子网络）。
- [ ] Quartz前端-图谱-聚合节点：增加聚合节点，按指定聚合字段（属性、标签或日期范围）聚合同类节点，显示每个聚合节点包含的子节点数量，同时聚合节点可以展开。
- [ ] Quartz前端-图谱-追踪展示：
  1. 关联节点深度/广度控制：用户点击某个节点，默认展示与当前节点直接相关的m个节点、间接相关的n层节点（m、n可配置），可按需点击按钮加载更广/更深关联节点。
  2. 关联追溯：用户可以不停地点击追溯关联关系。
  3. 详情列表组件：图谱下方增加列表区域，展示与当前节点直接相关的节点的详情，支持点击跳转至对应页面或高亮图谱中对应节点。支持分批加载/虚拟滚动以优化长列表渲染。
  （单独制作页面，支持与U3系统内功能嵌套关联使用，通过API接受图谱Json数据。）
- [x] Quartz前端-全文检索优化：支持复合检索（同时包含多个标签 `#` 或属性`@`）；添加`-`符号支持，用于排除指定内容。
- [ ] Quartz前端-反向链接优化：
  1. 支持根据U3的配置调整排序。
  2. 按文件夹、聚合字段分组显示反向链接，可收起展开。
  3. 支持分批加载/虚拟滚动以优化长列表渲染。
- [ ] Quartz前端-目录与文件夹页优化：
  - 目录：支持根据U3的配置调整排序。
  - 文件夹页：支持根据U3的配置调整排序；支持分批加载/虚拟滚动以优化渲染。
