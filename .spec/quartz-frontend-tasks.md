## 📝 常用配置速查

### 反向链接组件总高度
**文件**：`quartz/components/styles/backlinks.scss`

```scss
.backlinks {
  // [HEIGHT-CONTROL] 关键：让反向链接组件在 sidebar 中占据所有剩余空间
  // 效果：自动向下撑满到页面底部边距，类似 explorer 的行为
  flex: 1 1 auto;
}

.backlinks > ul.backlinks-list {
  // [HEIGHT-CONTROL] 关键：在 .backlinks 内部占据剩余高度
  flex: 1 1 auto;
  // [HEIGHT-CONTROL] 关键：允许 flex item 被压缩，否则内容多时无法限制高度
  min-height: 0;
}
```

**原理**：
- `.backlinks` 通过 `flex: 1 1 auto` 在 sidebar 的 flex column 布局中**吃掉所有剩余空间**
- `.backlinks-list` 通过 `flex: 1 1 auto` + `min-height: 0` 在组件内部**撑满除标题外的全部高度**
- 列表滚动条始终限制在组件内部，**不会导致页面整体滚动**

**如需限制最大/最小高度**，在 `.backlinks-list` 上追加：
```scss
max-height: 400px;  /* 上限 */
min-height: 120px;  /* 下限，避免内容太少时缩得太小 */
```

### 反向链接分批加载阈值
**文件**：`quartz/components/Backlinks.tsx`

```ts
const defaultOptions: BacklinksOptions = {
  threshold: 20,  // 总数超过此值时启用分批加载和默认折叠
}
```
